# pandas-comandos-basicos

Este repositório contém um arquivo com comandos de cálculos estatísticos e análises básicas utilizando a biblioteca Pandas em Python.

No arquivo é criado inicialmente um dicionário com dados referentes a alunos, contendo nome, notas e gênero. Em seguida, são utilizados comandos simples da biblioteca Pandas para realizar:

- estatísticas descritivas
- contagem de valores
- filtragem de dados
- criação de novas colunas
- ordenação da tabela
